# 🗓️ Calendar TodoList (Frontend)
Live test : https://calendar-todolist-eight.vercel.app/

A sleek, modern planning experience built with **React**, **Tailwind CSS**, and **Framer Motion**. This application allows users to manage their daily tasks with an integrated calendar and secure authentication.

## 🚀 Features
* **Secure Authentication**: JWT-based login and registration with email verification.
* **Interactive Calendar**: View and manage tasks based on specific dates.
* **Dynamic To-Do List**: Create, update, and delete tasks with real-time UI updates.
* **Responsive Design**: Fully optimized for mobile and desktop using Tailwind CSS.
* **Global State Management**: Utilizes React Context API for user authentication and session persistence.

## 🛠️ Tech Stack
* **Library**: React 18
* **Styling**: Tailwind CSS
* **Routing**: React Router v6
* **API Client**: Axios
* **Deployment**: Vercel
